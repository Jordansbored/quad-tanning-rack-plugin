# QuadTannery Plugin

A Hytale mod that adds Double and Quad Tanneries with faster processing speeds and more slots.

## Features

### Double Tannery
- **2x processing speed** (2.5x when upgraded)
- 4 input slots, 4 output slots
- Crafted from 2x Tannery at a Workbench
- Upgrade cost: 10x each of Light, Medium, Heavy Leather and Sturdy Chitin

### Quad Tannery
- **4x processing speed** (~6.7x when upgraded)
- 8 input slots, 8 output slots
- Crafted from 2x Double Tannery at a Workbench
- Upgrade cost: 20x each of Light, Medium, Heavy Leather and Sturdy Chitin

## Installation

1. Download the latest `QuadTannery-x.x.x.jar` from [Releases](https://github.com/Jordansbored/quad-tanning-rack-plugin/releases)
2. Place the JAR in your Hytale mods folder:
   - **Linux (Flatpak)**: `~/.var/app/com.hypixel.HytaleLauncher/data/Hytale/UserData/Mods/`
   - **Windows**: `%APPDATA%/Hytale/UserData/Mods/`
3. Launch Hytale and start a server with mods enabled

## Commands

- `/tanneries` - Plugin info command

## Requirements

- Hytale (Early Access)
- Java 25

## Building from Source

```bash
./gradlew build
```

The JAR will be in `build/libs/`.
